
To start: (test_site branch)

1. conda create -n teradata_intermediate python=3.7

2. conda activate teradata_intermediate
 
3. conda install flask

4. create flaskblog.py

5. Paste code from https://palletsprojects.com/p/flask/

6. Locate to directory, 'set FLASK_APP=flaskblog.py' and 'set FLASK_DEBUG=1'

7. flask run
